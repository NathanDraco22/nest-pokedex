# Ejecutar para Desarrollo

1. Clonar el repo
2. Ejecutar
```
npm install
```
3. Es necesario el Nest CLI
```
npm i -g @nestjs/cli
```
4. Levantar base de datos (docker)
```
docker-compose up -d
```
## Stack Usado

* MongoDB
* NestJS
* Docker


usar el paquete `@nest\mongoose`  tambien instalar `mongoose`

tambien instalar `nestjs/serve-static`
